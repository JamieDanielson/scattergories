The Game of Scattergories

Objective:

Score the most points by guessing words for each category that start with the letter rolled on the die

Gameplay

Everyone has the same list of categories.
A die is rolled to determine the letter for the round.
A 3 minute timer is set.
Guess as many words as you can in that 3 minute round.

Scoring:

  • +1 point for each word guessed (valid dictionary word)
  • +1 point for each unique word (nobody else guessed the same word)
  • +1 point for each double-word using category letter (e.g. Mickey Mouse if letter is M, category is famous character)
